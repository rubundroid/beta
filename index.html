ഉപയോഗക്രമം (Help Guide)

  • Step 1: സ്ഥാപനവിവരങ്ങളും ബജറ്റും നൽകുക.
  • Step 2: ആവശ്യമായ മരുന്നുകളുടെ എണ്ണം (Qty) രേഖപ്പെടുത്തുക.
  • Step 3: 'AUTO FILL' ബട്ടൺ ഉപയോഗിച്ച് ബജറ്റ് തുകയിലേക്ക്‌ മരുന്നുകളുടെ എണ്ണം ക്രമീകരിക്കാം.
  • Step 4: 'SAVE COMBINED / SPLIT PDF' ബട്ടൺ വഴി ഇൻഡന്‍റ് ഡൗൺലോഡ് ചെയ്യാം.

ഇൻഡന്‍റ് മേക്കർ - സഹായി

1. സ്ഥാപനത്തിന്‍റെ വിവരങ്ങൾ നൽകുക (Header Section)

  • • Institution Name & Address: സ്ഥാപനത്തിന്‍റെ പേരും വിലാസവും കൃത്യമായി നൽകുക. ഇത് പിഡിഎഫിന്റെ മുകളിൽ വിലാസമായി വരും.
  • • Indent Type: ഇൻഡന്‍റ് ഏത് വിഭാഗത്തിൽ പെടുന്നു എന്ന് രേഖപ്പെടുത്തുക (ഉദാ: Term-1, LSGD, NAM).
  • • Total Budget: അനുവദിച്ചിട്ടുള്ള ആകെ തുക ഇവിടെ ടൈപ്പ് ചെയ്യുക.

2. ഇൻഡന്റുകൾ JSON ആയി സേവ് ചെയ്യാനും ലോഡ് ചെയ്യാനും

നിങ്ങൾ തയ്യാറാക്കുന്ന ഇൻഡന്റുകൾ JSON ആയി ഡൌണ്‍ലോഡ് ചെയ്തു സൂക്ഷിച്ചു വെക്കാം. ഒരേ സമയം ഒന്നിൽ കൂടുതൽ ഇൻഡന്റുകൾ തയ്യാറാക്കാൻ വിവിധ ബ്രൗസറുകൾ ഉപയോഗിക്കാം.

  • • JSON: ഒരു ഫയൽ പേര് നൽകി JSON ബട്ടൺ അമർത്തുക.
  • • LOAD: JSON സേവ് ചെയ്തവ ലിസ്റ്റിൽ നിന്നും എടുക്കാം.
  • • DEL: JSON ആവശ്യമില്ലാത്തവ നീക്കം ചെയ്യാം.
  • • IMPORT: JSON സേവ് ചെയ്തവ വീണ്ടും ഉപയോഗിക്കാം.

3. മരുന്നുകൾ തിരഞ്ഞെടുക്കാൻ

  • • 🔍 Search: മരുന്നിന്റെ പേര് ടൈപ്പ് ചെയ്ത് കണ്ടെത്താം.
  • • 🔍 Rate Search: വിലയുടെ അടിസ്ഥാനത്തിൽ തിരയാം (ഉദാ: 40-50).
  • • TYPE: വിഭാഗം തിരിച്ച് (അരിഷ്ടം, തൈലം) കാണാൻ.

4. ബജറ്റ് ക്രമീകരണം (Smart Tools)

  • • AUTO FILL: ബജറ്റിന് അനുയോജ്യമായ രീതിയിൽ എണ്ണം ക്രമീകരിച്ച് ബാലൻസ് പൂജ്യമാക്കാൻ സഹായിക്കുന്നു.
  • • SHOW SELECTED: തിരഞ്ഞെടുത്ത മരുന്നുകൾ മാത്രം കാണാൻ.

5. പിഡിഎഫ് റിപ്പോർട്ട്

  • • SAVE COMBINED PDF: എല്ലാ മരുന്നുകളുടെയും ഒരുമിച്ചുള്ള PDF ഡൌണ്‍ലോഡ് ചെയ്യുന്നതിന്.
  • • SAVE SPLIT PDF: കഷായ ചൂർണ്ണങ്ങളും മറ്റുള്ളവയും പ്രത്യേകം PDF ഡൌണ്‍ലോഡ് ചെയ്യുന്നതിന്.

6. റീസെറ്റ് (Reset)

പുതിയ ഇൻഡന്‍റ് തുടങ്ങാൻ RESET ബട്ടൺ ഉപയോഗിക്കാം.

💡 പ്രായോഗിക നിർദ്ദേശങ്ങൾ:
• ഓരോ മാറ്റവും തനിയെ സേവ് (Draft) ചെയ്യപ്പെടുന്നു.
• AUTO FILL ഉപയോഗിക്കുന്നതിന് മുൻപ് കുറഞ്ഞത് രണ്ട് മരുന്നുകളെങ്കിലും തിരഞ്ഞെടുക്കുക.
